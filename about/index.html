骇客猫，一个中二程序猿。

平时喜欢逛些论坛诸如 Stack Overflow、 HackerNews 、 segmentfault 、 v2ex 、 乌云 、 知乎 等，同时还是 sae 认证的初级开发者。经过一年多的 web 开发的学习，掌握了一点基础的前端技术 以及 较为熟练的后端开发技术，其中后端开发主要用到的语言有 python && php ，我对 php 是十分热爱的，感觉是惊为天人的一门语言，非常的便捷优雅，之前都是用原生的 php 进行开发，喜欢自己造轮子。目前正在学习 php 的 Laravel 框架。 而 python 也吸引我在于他有好多第三方的库，语义化明显。目前我可以较熟练的使用 python 的 Django 框架，有 Django web 后端开发经验。

在编程之外，我还喜欢刷些 Dribbble 、 Pinterest 等设计师论坛，对其中精妙的设计十分着迷，对交互设计也有极大的兴趣，感觉一个应用是为人服务的，应该贴切人类的使用习惯并遵循人类优质的审美风格。

同时，我还是漫威巨粉，对一切超级英雄有着不切实际的幻想，常想去追逐他们风中飘忽不定的脸。

目前正在找实习 T.T 我的简历
