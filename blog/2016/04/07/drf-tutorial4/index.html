验证与授权

目前来看，我们的 API 并没有权限上的限制(即任何人都可以编辑或删除我们的 Movies )，这不是我们想要的。所以我们需要在 API 上做些限制以确保:

  • Movies 与 Users 关联起来。
  • 只有授权了的用户才能创建新的 Movies。
  • 只有 Movies 的创建者才可以更新或删除它。
  • 未授权的用户只能进行查看。
