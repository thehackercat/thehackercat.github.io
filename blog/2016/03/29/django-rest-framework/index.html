背景

Django REST Framework (以下简称 DRF )是一个轻量级的库，熟悉 Django 的话可以很容易的用它来构建 Web API。

安装前提

Django REST Framework 安装需要以下前提:

  • Python (2.7, 3.2, 3.3, 3.4, 3.5)
  • Django (1.7+, 1.8, 1.9)

我自己的环境是:

  • Python 2.7.10
